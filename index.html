Hello, world!

My name is Alina and I'm a Software and Web Developer from Germany.
I started working as a backend developer but now I’m working as a frontend developer.
I love HTML, CSS, Javascript and Angular.

This page is an information platform for beginners and tech interested people.
I'd like to share my experience and start into the tech industry and I try to help others with their step into this industry.
You can find some resources that help me to improve my coding skills everyday like recommendations on online courses, books, tools, ...


How did I get into tech?

I have always been interested with computers, video games and web design.
The first time I got in touch with programming was on social media platforms like myspace and chat rooms where you were able to create individual pages with plain HTML and inline CSS styling.
Although I chose another path after school and studied Event & Tourism Management. I found my way (back) into the tech industry.
After my studies I started new and began an internship and an apprenticeship afterwards in the IT department of the company I’m still working at.
During that time I also started to study Computer Science beside my daily full-time job. Like I mentioned before I started as a backend developer learning C# and Java.
At the end of my apprenticeship I also had the chance to learn the basics of web development including HTML, CSS and Javascript.
At the beginning of this year I joined an Angular in five days course and I immediately fell in love with it.
